Quasi-proxy
===========

Простенький "прокси" сервер для отображения модифицированных страниц Хабра.


Requirements
------------
Python3.5+


Installation
------------

```shell
    $ pip install -U -r requirements.txt
``` 

Run
---
```shell
    $ python quasi_proxy.py
```
Сервер будет доступен по адресу `127.0.0.1:1234`
